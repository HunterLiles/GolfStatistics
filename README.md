An app that devloped to track all my rounds of golf, and the courses I played at.

I currently keep track of:
* Each round score
* Date of each round (if no date is provided then I auto to todays date)
* Name of each course
* A description for each course

I am using StreamLit currently to make a prototype webpage and I am hosting it through StreamLit cloud.
  - I plan on switching over to an entirely HTML/JavaScript webpages but this gives me a way to start using the database and getting more ideas.
  - In the StreamLit webpages I can see the tables for the Courses and Rounds, I can calculate the handicap, and I can add/delete rounds.

As this DB grows I will continue to update what I have in it.
